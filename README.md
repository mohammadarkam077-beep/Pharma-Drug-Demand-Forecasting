# AI-Powered Pharma Drug Demand Forecasting & Decision Optimization System

An end-to-end pharmaceutical demand forecasting and prescriptive decision support system built with Python, Streamlit, forecasting models, Bayesian uncertainty analysis, scenario simulation, inventory planning, and investment optimization.

This project goes beyond traditional demand forecasting. It does not only predict future demand — it helps decision-makers decide:

- Which states should receive aggressive investment?
- Which states should be expanded selectively?
- Which states should be maintained or deprioritized?
- How much inventory should be prepared?
- Which states should be selected under a limited investment budget?
- How do decisions change under demand, revenue, or risk scenarios?
- What improvement is required for a state to move into a better decision category?

---

## Project Overview

Pharmaceutical demand planning is a high-impact business problem. Incorrect demand estimation can lead to:

- Stockouts
- Overproduction
- Wasted inventory
- Poor market allocation
- Missed revenue opportunities
- Inefficient field-force deployment

This system solves the problem using a complete analytics pipeline:
